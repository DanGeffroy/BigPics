## Techs

- css framework : don't know yet

The project will be hosted on github pages.
## Setup: 
Install angular cli
```bash
npm install -g angular-cli
```
Install the dependencies : 
```bash
npm install
```
Serve the app : 
```bash
ng serve
```
The app is now available on http://localhost:4200/
